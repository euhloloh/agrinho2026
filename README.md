# 🌱 Agrinho 2026

## 🌍 Tema
**Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente**

---

## 📌 Assunto Escolhido
**Agricultura de Precisão: tecnologia para produzir mais com menor impacto ambiental**

---

## 📖 Sobre o Projeto
Este projeto foi desenvolvido para o Concurso Agrinho 2026 na categoria Programação Web com Inteligência Artificial.

O site apresenta como a tecnologia pode ser aplicada no campo para aumentar a produtividade agrícola, reduzir desperdícios e preservar o meio ambiente, promovendo um futuro mais sustentável.

---

## 🎯 Objetivo
Demonstrar a importância da agricultura de precisão como ferramenta para equilibrar produção de alimentos e preservação ambiental, utilizando tecnologia como drones, sensores e GPS.

---

## 💻 Tecnologias Utilizadas
- HTML5  
- CSS3  
- JavaScript  
- Inteligência Artificial (apoio no desenvolvimento)

---

## ⚙️ Funcionalidades
- Página informativa sobre agricultura de precisão  
- Design responsivo (celular e computador)  
- Seções organizadas com conteúdo educativo  
- Quiz interativo em JavaScript  
- Botão de voltar ao topo
